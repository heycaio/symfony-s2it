# Symfony 2 S2IT



## Installation


```bash
- Clone this project git clone https://github.com/gammasoft/brasil-exemplo.git
- php composer update
- php app/console doctrine:database:create
- php app/console doctrine:schema:update
- Permission app/cache, app/logs
```
